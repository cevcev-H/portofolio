Tentang Saya

Halo! Saya adalah seorang lulusan SMK Yadika 3 yang memiliki ketertarikan besar di dunia teknologi, khususnya pengembangan game (Game Development).

Di waktu luang, saya sangat hobi membaca novel. Kebiasaan ini membantu saya dalam mengasah imajinasi, menyusun alur cerita, dan membangun konsep dunia yang kreatif—sesuatu yang sangat berguna ketika saya merancang sebuah game.

Minat utama saya saat ini adalah menciptakan game, mulai dari menyusun logika coding hingga mekanisme permainannya. Saya punya impian besar dan ambisi kuat untuk terus belajar agar suatu saat nanti bisa bergabung dan berkontribusi dalam project game skala besar (AAA/Big Project).

Berikut adalah beberapa tools dan teknologi yang sedang saya pelajari dan gunakan:

Game Logic HTML5 & CSS3 JavaScript Unity / Unreal Engine / Godot Git & GitHub
